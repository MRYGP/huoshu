# 决策记录 (ADR风格)

记录重要产品决策及其**理由**。

## ADR格式

每个决策记录包含：
- **背景** (Context) - 为什么需要做这个决策
- **决策** (Decision) - 做了什么决定
- **理由** (Rationale) - 为什么这样决定
- **后果** (Consequences) - 这个决策的影响
